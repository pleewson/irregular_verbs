# irregular_verbs
Console program that test your skills in irregular verbs.
<br> Fill the missing fields with correct answers or just read all irregular verbs.

<br> Programs:
<br> 1. It prints all irregular verbs with Polish translation.
<br> 2. Program prints first irregular verb then user will be prompt to fill second and third field.
<br> 3. Choose amount of irregular verbs to practice and fill second and third field.
